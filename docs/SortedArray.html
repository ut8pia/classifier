Class: SortedArray

SortedArray(comparatoropt, itemsopt)

The SortedArray is a straightforward Queue that employs a couple of properties:

  • items
  • comparator

to arrange items in an internal array based on a specific comparator.

You can locate various frequently used comparators within the properties of the global object ORDER. Nevertheless, you can provide any custom comparator of your preference to the SortedArray constructor.

const queue = new SortedArray(ORDER.ASCENDING);

The static method:

  • logSearch()

is specifically designed for performing a logarithmic search within an array to locate a specified item.

The SortedArray plays a crucial role in facilitating comparator-based equivalences between objects. These equivalences serve as an essential foundation for establishing representation-based equivalences, which are intended to be achieved through the Classifier.

For a thorough exploration of the ideas surrounding comparator-based and representation-based equivalences among objects, please consult the booklet:

Venditti, R. (2023). Classifier.js: Idea, Design, and Use Cases. Amazon KDP

(static) logSearch(item, comparator, startopt, endopt) → {Array}

Returns a pair [index, result]. The index represents the position within the array where the item is either currently located or where it should be inserted if it's not found; result can be either the item at that position, if it matches the specified item, or undefined.
