Class: Collection

Collection()

The Collection is an iterable class that introduces several abstract methods designed to define the core functionality of a collection. They include:

  • n()
  • has(item)
  • add(item)
  • remove(item)
  • clear()
  • view()

Building upon these abstract methods, the Collection class implements concrete methods that are inherited by any collection extending the class. These concrete methods include:

  • isEmpty()
  • let(item)
  • letAll(items)
  • removeAll(items)
