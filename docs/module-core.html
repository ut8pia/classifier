Module: core

The core package comprises two abstract classes:

  • Collection
  • Queue

These classes are essential in establishing the core principles of both a generic collection and a sorted collection. They do this by introducing abstract methods that are designed to be implemented concretely by the various collections available in the library. These methods form the foundation of functionality for the entire library.
