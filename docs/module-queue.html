Module: queue

The queue package encompasses classes that provide concrete implementations of the Queue interface. Among these, there are two classes, namely:

  • ArrayQueue
  • SortedArray

which internally manage an array of items.
Conversely, the remaining three classes:

  • Classifier
  • CommonCollection
  • TrueSet

internally utilize a Classifier.
