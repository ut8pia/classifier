Class: Queue

Queue()

The Queue class is an extension of the Collection class and acts as a template for declaring two essential methods that every sorted collection should encompass:

  • peek()
  • poll()

The class subsequently offers concrete implementations based on these abstract methods, which are inherited by all sorted collections:

  • select()
  • search()

The latter method effectively embodies the "search and selection" pattern that has inspired the development of the library.

To delve into a comprehensive discussion regarding this pattern and the related concept of:

  • Search space

please refer to the booklet:

Venditti, R. (2023). Classifier.js: Idea, Design, and Use Cases. Amazon KDP

search(space) → {Each}

Explores the given space by using this Queue as a stack. Each cycle, the first item of this queue is extracted and all the contiguous items returned by the search space are added. Items exceeding the max allowed length are removed.

The returned search is as instance of the Each class. Such a class, defined in the library @ut8pia/fluent, is intended to handle formal definitions of iterations. Its methods do not actually execute the iterations. This way, even an infinite search can be defined, restricted, transformed and finally resolved.

Parameters:

space function
